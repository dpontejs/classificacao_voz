# classificacao_voz
Este projeto tem como objetivo implementar um sistema classificador de comandos de voz. Para isso, foi utilizado um modelo baseado em redes neurais profundas com arquitetura Wav2Vec2, aplicado à base de dados Speech Commands (https://arxiv.org/abs/1804.03209). O modelo implementado alcança 95.37% de acurácia no conjunto de teste.
